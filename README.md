# Password Generator 🔐

A simple password generator built with **React** and **Vite**.  

## Features ✨
- Generate strong, random passwords.
- Customize password length and include/exclude:
  - Uppercase letters
  - Numbers
  - Special characters

## Tech Stack 🛠
- **React** (Frontend)
- **Vite** (Build tool)
- **CSS** for styling
